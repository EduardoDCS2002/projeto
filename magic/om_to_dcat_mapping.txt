# Mapeamento OpenMetadata → DCAT JSON-LD

fullyQualifiedName   → dct:identifier
name                 → dct:title
description          → dct:description
createdAt            → dct:issued
updatedAt            → dct:modified

owner.name ou owner.displayName
                     → dct:publisher.foaf:name

tags[].tagFQN        → dcat:keyword[]

columns[]            → om:columns[] (extensão)
tableType            → om:tableType
profile              → om:profile
usageSummary         → om:usageSummary

service.name         → om:service
serviceType          → om:serviceType
database.name        → om:database (informativo)
databaseSchema.name  → om:schema

sourceUrl            → dcat:distribution[0].dcat:accessURL OR om:sourceUrl
owner (completo)     → om:owner (como objeto)
id                   → om:id (preservado apenas como metadado auxiliar)

# Campos que não têm correspondência direta no DCAT:
- relações com lineage ou entidades relacionadas
- comentários, votos, threads
- métricas internas (fidelidade pode ser adicionada como "om:*")
