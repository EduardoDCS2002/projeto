# Mapeamento DCAT → CKAN

dct:identifier       → name
dct:title            → title
dct:description      → notes
dct:issued           → Ignorado (CKAN gera metadata_created)
dct:modified         → Ignorado (CKAN gera metadata_modified)

dct:publisher.foaf:name
                     → owner_org (precisa existir previamente no CKAN)

dcat:keyword[]       → tags[] (convertido para: [{ "name": "..." }])

dcat:distribution[]  → resources[]
  - dct:title        → name
  - dcat:accessURL   → url
  - dcat:mediaType   → mimetype
  - dct:description  → description
  - dct:issued       → created
  - dct:modified     → last_modified

ckan:author          → author
ckan:author_email    → author_email
ckan:maintainer      → maintainer
ckan:maintainer_email → maintainer_email
ckan:license_id      → license_id
ckan:isopen          → isopen
ckan:state           → state
ckan:extras[]        → extras[]
ckan:groups[]        → groups[]
ckan:relationships_as_object[] → relationships_as_object[]
ckan:relationships_as_subject[] → relationships_as_subject[]

# Campos não utilizados diretamente:
- @context, @type → ignorados pela API CKAN
