# Mapeamento CKAN → DCAT JSON-LD

name                 → dct:identifier
title                → dct:title
notes                → dct:description
metadata_created     → dct:issued
metadata_modified    → dct:modified

organization.title   → dct:publisher.foaf:name

tags[].name          → dcat:keyword[]

resources[]          → dcat:distribution[]
  - name             → dct:title
  - url              → dcat:accessURL
  - mimetype         → dcat:mediaType
  - description      → dct:description
  - created          → dct:issued
  - last_modified    → dct:modified

author               → ckan:author
author_email         → ckan:author_email
maintainer           → ckan:maintainer
maintainer_email     → ckan:maintainer_email
license_id           → ckan:license_id
isopen               → ckan:isopen
state                → ckan:state
extras[]             → ckan:extras[]
groups[]             → ckan:groups[]
relationships_as_object[] → ckan:relationships_as_object[]
relationships_as_subject[] → ckan:relationships_as_subject[]

# Campos adicionados como extensões:
- ckan:* são incluídos como campos customizados fora do vocabulário DCAT
